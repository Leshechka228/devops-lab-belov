# DevOps Lab - belov

Проект для лабораторной работы по GitHub Webhooks.

## Структура проекта
devops-lab/
├── index.html # Основная страница сайта
├── nginx.conf # Конфигурация веб-сервера
├── deploy.sh # Скрипт автоматического развертывания
├── test.sh # Тестирование сайта
├── webhook-server.py # Webhook сервер
├── install-nginx.sh # Скрипт установки nginx
├── install-frp.sh # Скрипт установки FRP
└── README.md # Документация
